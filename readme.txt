Plugin para integración de Improve Bot para Wordpress

== Description ==

Plugin para integración de [Improve Bot](https://improvebot.krino.ai/ "Improve Bot") para Wordpress

== Installation ==

1. Activa el plugin en la ventana de Plugins.
2. Agrega el API KEY de tu Bot dado por Improve Bot.
